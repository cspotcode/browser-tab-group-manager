# browser-tab-group-manager

Simple UI for viewing a one-page inventory of all open windows, tabs, and tab groups.

Not published to extension marketplace; meant for local usage in Developer mode.

> [!NOTE]  
> Code was written with AI assistance and minimal review. I couldn't justify spending more time on this.

## Installation

Dev workflows described in `justfile`

`just build` or `just watch` to build the code.

Enable developer mode in your browser, then install "dist/" subdirectory as an unpacked extension.
